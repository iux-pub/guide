   ```scss
   // 반응형 미디어쿼리 믹스인 (D-09)
   // D-04: 3단계 — 모바일(0~767px), 태블릿(768px~1279px), PC(1280px~)
   // tablet-up: 태블릿+PC 공통 스타일 적용 시 사용 — D-04의 3단계 구분에서
   //   태블릿과 PC가 동일한 레이아웃(12컬럼)을 공유하는 경우가 빈번하므로
   //   편의를 위해 추가. min-width: $breakpoint-tablet 이상 전체를 커버.
   @use '../1-settings/breakpoints' as bp;

   @mixin respond-to($device) {
     @if $device == 'mobile' {
       @media (max-width: #{bp.$breakpoint-tablet - 1px}) {
         @content;
       }
     } @else if $device == 'tablet' {
       @media (min-width: #{bp.$breakpoint-tablet}) and (max-width: #{bp.$breakpoint-pc - 1px}) {
         @content;
       }
     } @else if $device == 'tablet-up' {
       @media (min-width: #{bp.$breakpoint-tablet}) {
         @content;
       }
     } @else if $device == 'pc' {
       @media (min-width: #{bp.$breakpoint-pc}) {
         @content;
       }
     }
   }
   ```

3. **`src/scss/2-tools/_functions.scss`** — sass-rem 설정 (SCSS-04):
   ```scss
   // REM 변환 함수 — sass-rem v4
   // 사용법: rem.convert(16px) → 1rem
   // 사용법: rem.convert(8px 16px) → 0.5rem 1rem (shorthand 지원)
   @use 'sass-rem' as rem;

   // sass-rem의 rem.convert()를 @forward로 재노출
   // 다른 파일에서 @use '2-tools' as *; 후 rem.convert() 사용
   ```
   주의: sass-rem v4의 `@use` 호환성 확인 필요. `@use 'sass-rem'`이 동작하지 않으면 `@use 'sass-rem/sass-rem'` 또는 node_modules 경로 시도. 최종적으로 안 되면 직접 rem 함수 작성:
   ```scss
   @function rem-convert($values) {
     $result: ();
     @each $value in $values {
       @if type-of($value) == 'number' and unit($value) == 'px' {
         $result: append($result, $value / 16px * 1rem);
       } @else {
         $result: append($result, $value);
       }
     }
     @return $result;
   }
   ```
