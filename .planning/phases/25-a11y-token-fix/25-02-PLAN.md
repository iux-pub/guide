**포커스 링 통일 (A11YFIX-06):**

`&__input, &__select, &__textarea` 블록 내부의 기존 `:focus` 스타일 3곳을 모두 `:focus-visible` + outline 패턴으로 교체한다.

1) 기본 포커스 (48-53행 부근):
변경 전:
```scss
&:focus {
  outline: none;
  border-color: var(--color-primary);
  box-shadow: 0 0 0 2px rgb(37 110 244 / 20%);
  box-shadow: 0 0 0 2px color-mix(in srgb, var(--color-primary) 20%, transparent);
}
```
변경 후:
```scss
&:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
  border-color: var(--color-primary);
}
```

2) --error 상태 포커스 (65-68행 부근):
변경 전:
```scss
&:focus {
  box-shadow: 0 0 0 2px rgb(222 52 18 / 20%);
  box-shadow: 0 0 0 2px color-mix(in srgb, var(--color-danger) 20%, transparent);
}
```
변경 후:
```scss
&:focus-visible {
  outline: 2px solid var(--color-danger);
  outline-offset: 2px;
}
```

3) --success 상태 포커스 (75-78행 부근):
변경 전:
```scss
&:focus {
  box-shadow: 0 0 0 2px rgb(34 135 56 / 20%);
  box-shadow: 0 0 0 2px color-mix(in srgb, var(--color-success) 20%, transparent);
}
```
변경 후:
```scss
&:focus-visible {
  outline: 2px solid var(--color-success);
  outline-offset: 2px;
}
```

**reduced-motion 적용 (A11YFIX-05):**

`&__input, &__select, &__textarea` 블록의 `transition: border-color var(--transition-fast);` 선언 바로 아래에 추가:
```scss
// 접근성: 모션 감소 설정 대응
@media (prefers-reduced-motion: reduce) {
  transition-duration: 0.01ms;
}
```

주의:
- box-shadow 포커스를 완전히 제거하고 outline으로 교체한다 (outline은 고대비 모드에서도 보임)
- `outline: none` 선언을 제거한다
- --error와 --success의 border-color 선언은 유지하고, 포커스 시 outline-color만 변경한다
