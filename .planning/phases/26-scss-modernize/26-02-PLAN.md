**1. modal.js 수정 — modal--active 클래스 토글:**

현재 openModal 함수에서 `modal.setAttribute('aria-hidden', 'false')` + `modal.style.display = 'flex'` 하고 있다. 여기에 `.modal--active` 클래스를 추가한다:

openModal 함수 내부, `modal.style.display = 'flex'` 바로 아래에 추가:
```javascript
modal.classList.add('modal--active')
```

closeModal 함수 내부, `modal.style.display = 'none'` 바로 위에 추가:
```javascript
modal.classList.remove('modal--active')
```

**2. modal.scss 수정 — 열림 애니메이션 (MOTION-01):**

파일 맨 아래 (`.modal { ... }` 블록 닫힌 후)에 `@keyframes` 정의:
```scss
// 모달 열림 애니메이션 (MOTION-01)
@keyframes modal-open {
  from {
    opacity: 0;
    transform: scale(0.95);
  }

  to {
    opacity: 1;
    transform: scale(1);
  }
}
```

`.modal` 블록 내부에서 `&--active` 모디파이어 추가 (`&[aria-hidden='false']` 바로 아래):
```scss
// 활성 상태 — 열림 애니메이션 적용
&--active &__container {
  animation: modal-open 0.3s var(--transition-base) both;
}

// 활성 상태 — 오버레이 페이드인
&--active &__overlay {
  animation: modal-open 0.3s ease both;
}
```

주의: `animation: modal-open 0.3s ease both;`를 사용한다. `var(--transition-base)`는 트랜지션용이므로 animation에는 `ease` 직접 사용.

**중요 수정**: animation 축약형에서 duration은 0.3s (300ms)를 사용한다.

기존 `prefers-reduced-motion: reduce` 블록이 `transition-duration: 0.01ms`만 처리하고 있다. animation도 비활성화하려면 `.modal` 블록 내부의 기존 reduced-motion 블록 근처에 추가:

오버레이의 기존 reduced-motion 블록 안에 `animation: none;` 추가:
```scss
@media (prefers-reduced-motion: reduce) {
  transition-duration: 0.01ms;
  animation: none;
}
```

`&__container`에도 reduced-motion 블록 추가 (현재 없음):
```scss
&__container {
  // ... 기존 스타일 ...

  // 접근성: 모션 감소 설정 대응
  @media (prefers-reduced-motion: reduce) {
    animation: none;
  }
}
```
