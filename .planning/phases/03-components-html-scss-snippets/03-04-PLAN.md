## 기본 마크업

**구체적 내용 (컴포넌트별):**

**btn.md:**
- 기본 마크업: `<button type="button" class="btn btn--primary">버튼 텍스트</button>`
- Variant: primary, secondary, outline, text, ghost, link
- 크기: sm (32px), 기본 (40px), lg (48px)
- 접근성: `<button>` 태그 사용 필수, 아이콘만 버튼은 `aria-label` 필수, 비활성은 `disabled` 속성, 최소 터치 영역 44px

**form.md:**
- 기본 마크업: label + input 그룹
- Variant: input, select, textarea, checkbox, radio, required, error, success, disabled
- 접근성: `<label for="">` 필수 연결, 에러 시 `aria-invalid="true"` + `aria-describedby`, 에러 메시지에 `role="alert"`, 필수 필드 `required` 속성

**card.md:**
- 기본 마크업: `<article class="card">` + header/body/footer
- Variant: 기본, horizontal, image, featured
- 접근성: `<article>` 시맨틱 태그, 이미지에 `alt` 필수

**table.md:**
- 기본 마크업: caption + thead/th + tbody/td
- Variant: 기본, striped, bordered, 반응형 (wrapper)
- 접근성: `<caption>` 필수 (sr-only 가능), `<th scope="col">` 또는 `scope="row"` 필수, 빈 상태 `<td colspan>` 안내 텍스트

**modal.md:**
- 기본 마크업: trigger button + modal (role="dialog")
- 구성: overlay + container + header/body/footer + close button
- 접근성: `role="dialog"` + `aria-modal="true"` + `aria-labelledby` + `aria-hidden`, ESC 닫기, 포커스 트랩, 트리거로 포커스 복귀, `body overflow: hidden`
- JS: `<script src="path/to/modal.js"></script>` 필수

**tab.md:**
- 기본 마크업: tablist + tab buttons + tabpanels
- 구성: tab__list + tab__button + tab__panel
- 접근성: `role="tablist"` + `role="tab"` + `role="tabpanel"`, `aria-selected`, `aria-controls` + `aria-labelledby`, 비활성 탭 `tabindex="-1"`, 비활성 패널 `hidden`
- 키보드: ArrowLeft/Right, Home/End
- JS: `<script src="path/to/tab.js"></script>` 필수

**pagination.md:**
- 기본 마크업: `<nav aria-label="페이지 네비게이션">` + ul/li/a
- Variant: 기본, current, prev, next, disabled
- 접근성: `<nav>` + `aria-label`, 각 숫자 `aria-label="페이지 N"`, 현재 페이지 `aria-current="page"`, 이전/다음 `aria-label`

**breadcrumb.md:**
- 기본 마크업: `<nav aria-label="현재 위치">` + ol/li
- 구성: list + item + link + current
- 접근성: `<nav>` + `aria-label="현재 위치"`, `<ol>` 순서 목록, 현재 항목 `aria-current="page"`, CSS 구분자 (스크린리더 자동 무시)

**boilerplate.md:**
- 기본 마크업: 전체 HTML 구조 (lang="ko", viewport, skip-to-content, header/main/footer)
- 접근성: skip-to-content "본문 바로가기", `role="banner"` / `role="main"` / `role="contentinfo"`, `id="main-content"`
