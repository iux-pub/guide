Line Height

--leading-tight (1.2)
웹 접근성은 모든 사용자가 웹 콘텐츠에 접근하고 이용할 수 있도록 보장하는 것을 의미합니다. 시각, 청각, 운동 능력 등에 관계없이 누구나 정보를 얻을 수 있어야 합니다. WCAG 2.1 AA 기준을 준수하면 대부분의 사용자가 콘텐츠를 이용할 수 있습니다.
