<action>
1. `src/scss/6-components/_btn.scss` 생성. BEM `.btn` 블록:

기본 스타일:
- `display: inline-flex; align-items: center; justify-content: center`
- `gap: var(--spacing-xs)`
- `padding: var(--spacing-sm) var(--spacing-md)` (기본 높이 ~40px)
- `border: 1px solid transparent; border-radius: var(--radius-base)`
- `font-family: var(--font-family-base); font-size: var(--font-size-base); font-weight: var(--font-weight-bold)`
- `line-height: var(--leading-tight); text-decoration: none; cursor: pointer`
- `transition: background-color var(--transition-fast), border-color var(--transition-fast), color var(--transition-fast)`

Variant modifiers:
- `&--primary`: `background-color: var(--color-primary); color: var(--color-white)`. hover: `var(--color-primary-dark)`
- `&--secondary`: `background-color: var(--color-gray-100); color: var(--color-text)`. hover: `var(--color-gray-200)`
- `&--outline`: `border-color: var(--color-border); background-color: transparent; color: var(--color-text)`. hover: `background-color: var(--color-bg-secondary)`
- `&--text`: `background-color: transparent; color: var(--color-primary); border-color: transparent`. hover: `background-color: var(--color-bg-secondary)`
- `&--ghost`: `background-color: transparent; color: var(--color-text); border-color: transparent`. hover: `background-color: var(--color-gray-100)`
- `&--link`: `background-color: transparent; color: var(--color-primary); border-color: transparent; text-decoration: underline`. hover: `color: var(--color-primary-dark)`

Size modifiers:
- `&--sm`: `padding: var(--spacing-xs) var(--spacing-sm); font-size: var(--font-size-sm); min-height: 4.4rem` (터치 영역 44px 보장)
- `&--lg`: `padding: var(--spacing-md) var(--spacing-lg); font-size: var(--font-size-md)`

States:
- `&:disabled`: `opacity: 0.5; cursor: not-allowed`
- `&:focus-visible`: `outline: 2px solid var(--color-primary); outline-offset: 2px`
- 모든 hover에 `:not(:disabled)` 조건 추가

2. `src/scss/6-components/_index.scss`에 `@forward 'btn';` 추가

3. `npm run lint:css && npm run build:css` 실행 확인
</action>
