폼 (Form)

텍스트 입력

<div class="form__group">
  <label class="form__label" for="input-text">이름</label>
  <input type="text" id="input-text" class="form__input" placeholder="내용을 입력하세요">
</div>

필수 입력

<div class="form__group">
  <label class="form__label form__label--required" for="input-required">이메일</label>
  <input type="email" id="input-required" class="form__input" placeholder="내용을 입력하세요" required aria-required="true">
</div>

에러 상태

필수 입력 항목입니다. 
<div class="form__group">
  <label class="form__label form__label--required" for="input-error">이메일</label>
  <input type="email" id="input-error" class="form__input form__input--error"
    placeholder="내용을 입력하세요" aria-invalid="true" aria-describedby="error-msg">
  <span id="error-msg" class="form__message form__message--error" role="alert">
    필수 입력 항목입니다.
  </span>
</div>

성공 상태

입력이 확인되었습니다. 
<div class="form__group">
  <label class="form__label" for="input-success">이메일</label>
  <input type="email" id="input-success" class="form__input form__input--success" value="user@example.com">
  <span class="form__message form__message--success">입력이 확인되었습니다.</span>
</div>

비활성 (Disabled)

<div class="form__group">
  <label class="form__label" for="input-disabled">이름</label>
  <input type="text" id="input-disabled" class="form__input" value="비활성 입력" disabled>
</div>

Select 드롭다운

<div class="form__group">
  <label class="form__label" for="select-demo">부서 선택</label>
  <select id="select-demo" class="form__select">
    <option value="">선택하세요</option>
    <option value="ux">UX팀</option>
    <option value="dev">개발팀</option>
    <option value="plan">기획팀</option>
  </select>
</div>

Textarea

최대 500자까지 입력 가능합니다. 
<div class="form__group">
  <label class="form__label" for="textarea-demo">비고</label>
  <textarea id="textarea-demo" class="form__textarea" placeholder="내용을 입력하세요"></textarea>
  <span class="form__help">최대 500자까지 입력 가능합니다.</span>
</div>

</label>
