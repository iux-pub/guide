## 가이드 목록

| 가이드 | 설명 |
|--------|------|
| [시맨틱 HTML 마크업](/guides/semantic-html/) | heading 계층, section/article/nav/aside 사용 규칙 |
| [이미지 처리 규칙](/guides/image-optimization/) | 포맷 선택, lazy loading, SVG 최적화, 파일 네이밍 |
| [CSS 성능 가이드라인](/guides/css-performance/) | 선택자 깊이 제한, 미사용 CSS 감지, 파일 크기 기준 |
| [CSS 애니메이션/트랜지션](/guides/css-animation/) | 작성 규칙, prefers-reduced-motion 대응, GPU 가속 |
