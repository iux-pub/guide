    ### conventions 블록에 넣을 내용:

    **1. BEM 네이밍 규칙**
    - 필수 패턴: `.block-name`, `.block__element`, `.block--modifier`, `.block__element--modifier`
    - SCSS 작성: 반드시 `&__element`, `&--modifier` 중첩 사용
    - element 2단계 중첩 금지: `.card__header__title` -> `.card__title`로 평탄화
    - modifier는 block 또는 element에만 부착

    금지 패턴 (구체적으로):
    - `.btn-primary` -> `.btn--primary` (modifier에 `--` 필수)
    - `.card-header` -> `.card__header` (element에 `__` 필수)
    - `.input-box.error` -> `.form__input--error` (별도 클래스 조합 대신 modifier)
    - `.card-header h4` -> `.card__title` (요소 선택자 의존 금지)
    - 시각적 속성 이름 금지: `.btn-gray` -> `.btn--secondary` (의미적 이름 사용)

    **2. CSS 규칙**
    - BEM (Block__Element--Modifier) 필수
    - 인라인 스타일 사용 금지
    - `!important` 사용 금지 (부득이한 경우 주석 필수)
    - CSS Custom Properties(토큰) 우선 사용, 하드코딩 값 금지

    **3. 디자인 토큰 사용법**
    실제 토큰 파일에서 읽어온 변수명을 나열:
    - 색상: `var(--color-primary)`, `var(--color-gray-900)` 등 (실제 변수명 나열)
    - 타이포: `var(--font-size-base)`, `var(--font-size-lg)` 등
    - 간격: `var(--spacing-xs)` ~ `var(--spacing-3xl)`
    - 기타: `var(--radius-sm)`, `var(--shadow-sm)`, `var(--transition-fast)` 등
    - 하드코딩 예시와 토큰 사용 예시 대조

    **4. 린트 규칙**
    - `npm run lint:css` — SCSS 린트 실행
    - `npm run lint:css:fix` — 자동 수정
    - BEM 위반은 warning (빌드는 성공)
    - 코드 작성 후 반드시 lint:css 확인

    ### architecture 블록에 넣을 내용:

    **5. SCSS 구조 (ITCSS)**
    - 7레이어 구조와 각 레이어에 넣을 파일 종류
    - 새 컴포넌트: `src/scss/6-components/_컴포넌트명.scss`
    - 새 유틸리티: `src/scss/7-utilities/_유틸리티명.scss`
    - @use/@forward 모듈 시스템 (NOT @import)
    - style.scss에 새 레이어 @use 추가

    **6. 반응형 작성**
    - `@include respond-to(mobile)` / `tablet` / `tablet-up` / `pc`
    - 62.5% REM 트릭: 1rem = 10px, 따라서 16px = 1.6rem
    - 모바일 퍼스트 접근

    **7. 접근성 기본**
    - 이미지에 `alt` 속성 필수
    - 인터랙티브 요소에 `aria-label` 또는 텍스트 레이블 필수
    - 키보드 네비게이션 지원 필수
    - 색상 대비 4.5:1 이상
    - Swiper 등 슬라이더에 `aria-live`, `role` 속성 필수

    **8. 코딩 스타일**
    - 들여쓰기: 2 spaces
    - 따옴표: single quote
    - 세미콜론: 없음 (SCSS에서는 세미콜론 사용)
    - 주석은 한국어로 작성

    핵심: 모든 규칙은 명령형("~하라", "~사용하라", "~금지")으로 작성하여 AI가 지시문으로 바로 해석 가능하게 한다.
    토큰 변수명은 실제 파일에서 읽어온 정확한 값을 사용한다 (추측하지 말 것).
