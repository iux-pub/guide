이 가이드 시스템은 인포마인드 UX팀의 디자인 및 퍼블리싱 표준을 체계화한 것이다. 신규 팀원이 프로젝트에 빠르게 합류하고, 디자이너-퍼블리셔 간 핸드오프를 표준화하는 데 목적이 있다.

## 가이드 시스템 구성

| 영역 | 설명 |
|------|------|
| **ITCSS 7레이어** | SCSS를 Settings, Tools, Generic, Elements, Objects, Components, Utilities 7단계로 조직한다 |
| **BEM 네이밍** | 모든 CSS 클래스명을 Block__Element--Modifier 패턴으로 작성한다 |
| **디자인 토큰** | 색상, 간격, 타이포그래피 등 스타일 값을 CSS Custom Properties로 관리한다 |
| **접근성** | KWCAG/WCAG 2.1 AA 기준을 준수하여 공공기관 납품 요건을 충족한다 |
