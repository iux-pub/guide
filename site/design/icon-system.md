아이콘 크기, 라이브러리 선택, 접근성 규칙을 정의한다. 아이콘 시스템의 일관성은 UI 품질에 직접적 영향을 미치므로 이 규칙을 엄격히 따른다.

## 8pt 그리드 기반 크기

### 허용 크기

| 크기 | 용도 |
|------|------|
| 12px | 아주 작은 인라인 인디케이터 |
| 16px | 밀집된 UI, 소형 텍스트(12~13px) 옆 |
| 20px | 기본 -- 본문(14~16px) 옆 |
| 24px | 표준 아이콘 버튼, Nav 항목, 리스트 행 |
| 32px | 기능 아이콘, 섹션 헤딩 |
| 40px | Empty State 아이콘, 카드 장식 |
| 48px | 대형 Empty State, 온보딩 |

### 금지 크기

**18px, 22px, 26px, 28px, 36px** -- 광학 그리드 이탈로 어색하다. 이 크기들은 8pt 그리드에 정렬되지 않는다.

### 텍스트 크기 ↔ 아이콘 크기 페어링

| 텍스트 | 아이콘 |
|--------|--------|
| 12px | 12~14px |
| 14px | 16px |
| 16px | 20px |
| 18~20px | 24px |
| 24px+ | 28~32px |

## 컨텍스트별 크기

| 컨텍스트 | 아이콘 크기 | 비고 |
|----------|------------|------|
| 버튼 내 아이콘 | 16px | 텍스트와 함께 |
| 독립 아이콘 버튼 | 24px | 터치 타겟 44px 확보 |
| 대형 Empty State | 32~48px | 시각적 포인트 |
| 인라인 상태 표시 | 12~16px | 텍스트 옆 |

## 표준 라이브러리

### 권장 라이브러리

| 라이브러리 | 스타일 | 적합한 제품 |
|-----------|--------|------------|
| **Lucide** | 2px stroke, 깔끔한 outline | 현대적 SaaS, 개발자 도구 |
| **Phosphor** | 멀티웨이트, 범용 | 유연성 최고, 어디든 |
| **Tabler Icons** | Outline, 매우 풍부 | Enterprise, 관리 도구 |

**인포마인드 권장**: Lucide 또는 Tabler Icons (현재 프로젝트 선택을 따르되, **혼용은 금지**한다)

### 핵심 원칙

- **라이브러리 1종만 사용**: 라이브러리가 다르면 획 굵기, 광학 비율, 모서리가 달라 통일감이 깨진다
- **Outline/Filled 스타일 혼용 금지**: 한 제품 내에서 스타일을 통일한다
- **같은 아이콘 = 같은 의미**: 한 제품에서 ★이 "즐겨찾기"이면, 다른 곳에서 ★을 "평점"으로 사용하지 않는다

## 접근성

### 장식용 아이콘

텍스트와 함께 사용되어 의미를 보충하는 아이콘은 장식용으로 처리한다.

```html
<!-- 장식용: aria-hidden으로 숨긴다 -->
<button>
  <svg aria-hidden="true">...</svg>
  삭제하기
</button>
```

### 의미 있는 아이콘

텍스트 없이 단독으로 사용되는 아이콘은 반드시 `aria-label`을 제공한다.

```html
<!-- 의미 있는 아이콘: aria-label 필수 -->
<button aria-label="다이얼로그 닫기">
  <svg aria-hidden="true">...</svg>
</button>
```

### 모호한 아이콘 판별

| 범용 아이콘 (레이블 불필요) | 모호한 아이콘 (레이블 필수) |
|---------------------------|---------------------------|
| X 닫기, 메뉴, 검색, + 추가, 확인 | 즐겨찾기/평점, 알림/리마인더, 설정/구성, 파일/프로젝트 |

모호한 아이콘에는 반드시 **텍스트 레이블** 또는 **툴팁**을 함께 제공한다.

## 터치 타겟

아이콘 자체가 24px이라도 인터랙티브할 경우 **44x44px** 터치 타겟을 필수로 확보한다.

```css
.icon-button {
  width: 44px;
  height: 44px;
  display: flex;
  align-items: center;
  justify-content: center;
}
/* 아이콘은 내부에서 20~24px */
```

## 정렬

아이콘과 텍스트를 함께 쓸 때 Flexbox로 정렬한다.

```css
.icon-with-label {
  display: flex;
  align-items: center;
  gap: 8px;
}
/* 인라인 아이콘: vertical-align: -0.125em */
```

## Do / Don't

| Do | Don't |
|----|-------|
| 라이브러리 1종만 사용 | Outline/Filled 혼용 |
| 크기: 12/16/20/24/32/40/48px | 18/22/26/28/36px 사용 |
| 터치 타겟 44x44px 확보 | 24px 아이콘을 그냥 버튼으로 |
| 모호한 아이콘에 레이블/툴팁 | 레이블 없는 애매한 아이콘 |
| `aria-label` 아이콘 버튼 전체 | aria-label 없는 아이콘 버튼 |
| 같은 아이콘 = 같은 의미 | 맥락마다 다른 의미로 재사용 |
