HTML 페이지 보일러플레이트

기본 구조

모든 HTML 페이지의 기본 골격. 접근성 필수 요소(본문 바로가기, 시맨틱 태그, ARIA role)를 포함한다.

본문 바로가기

[header: 로고, GNB 영역]

[main: 페이지 콘텐츠 영역]

[footer: 푸터 영역]

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>페이지 제목</title>
  <link rel="stylesheet" href="../../dist/css/style.css">
</head>
<body>
  <a href="#main-content" class="skip-to-content">본문 바로가기</a>

  <header class="site-header" role="banner">
    <div class="container">
      <!-- 로고, GNB -->
    </div>
  </header>

  <main id="main-content" class="site-main" role="main">
    <div class="container">
      <!-- 페이지 콘텐츠 -->
    </div>
  </main>

  <footer class="site-footer" role="contentinfo">
    <div class="container">
      <!-- 푸터 -->
    </div>
  </footer>
</body>
</html>

필수 요소 체크리스트

  • 1. lang="ko" -- 문서 언어 선언
  • 2. meta viewport -- 반응형 뷰포트 설정
  • 3. .skip-to-content -- 본문 바로가기 링크 (접근성)
  • 4. role="banner" -- header 랜드마크
  • 5. role="main" -- main 랜드마크
  • 6. role="contentinfo" -- footer 랜드마크
  • 7. id="main-content" -- 본문 바로가기 타겟
  • 8. .container -- 콘텐츠 최대 너비 제한
