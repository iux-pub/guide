# Contributing

기여 가이드는 문서 사이트에서 확인하세요.

`npm run serve` 실행 후 **거버넌스 > 기여 가이드** 페이지를 참조하세요.

## 빠른 시작

```bash
git clone https://github.com/iux-pub/guide.git
cd guide
npm install --legacy-peer-deps
npm run build:css
npm run lint:css
npm run test:a11y
```
